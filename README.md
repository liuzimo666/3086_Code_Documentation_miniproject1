# CMML Mini-project 1

This repository contains the code and documentation for my CMML mini-project 1.

## Contents
- `COPASI_models/`: COPASI model files
- `model_data/`: exported simulation data
- `plot_all_figures.py`: Python script for generating figures
- `project1_documentation.ipynb`: Jupyter notebook documentation

## How to reproduce
1. Make sure all files remain in the same directory structure.
2. Run `plot_all_figures.py`.
3. The script will create a `figures/` folder automatically.

## Notes
The Python script uses relative paths, so the folder structure should not be changed internally.
